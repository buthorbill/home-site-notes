# home-site-notes

## 项目简介

`home-site-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的集中存放与版本管理空间。你可以将其视为一个轻量的页面集合仓库，用于保存、整理或展示各类自包含的网页内容。

## 目录说明

仓库的根目录结构如下（可能随更新调整）：

```
home-site-notes/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 页面引用的资源文件（如图片、样式、脚本）
├── index.html       # 可选：仓库入口页面，列出所有归档页面
└── README.md        # 本文件
```

- **pages/**：每个 HTML 文件对应一个独立页面，文件名建议使用有意义的英文或拼音，便于识别。
- **assets/**：若页面需要额外资源，可统一放置于此，按需组织子目录。
- **index.html**：非必需，可提供一个简单的导航页面，方便浏览所有归档内容。

## 页面归档说明

- 所有归档页面均为独立、自包含的 HTML 文件，不依赖外部服务或数据库。
- 页面内容可涵盖笔记、工具、演示、实验等多种用途，但不得包含任何违法、侵权或不当信息。
- 归档时请确保页面无外部引用（如跨域资源、外链），或已明确注明资源来源与许可。
- 建议在每个 HTML 文件头部添加简要注释，说明页面用途、创建日期及作者（可选）。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或改进现有页面。
- 提交前请确保页面在主流浏览器中可正常渲染，无明显的兼容性或功能性错误。
- 删除或修改已有页面时，请说明理由，避免破坏其他页面的引用关系。
- 本仓库使用 Git 进行版本管理，所有历史记录可追溯。
